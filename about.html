Export Exposure: Analyzing Turkey's Food Safety Challenges in EU Trade:

Description:

Tools Used:

  • Created dynamic visualizations that highlighted trends in food safety challenges and geographical distributions of risks.

 

Outcome:

Provided actionable insights into Turkeys food safety issues in EU trade, with data-driven recommendations for risk management and improvement.

LEED Certification Analysis: Performance of U.S. Projects:

Description:

Developed an analysis of LEED certifications across the United States, examining the impact of Gross Floor Area (GFA), certification levels, and project types using SQL for data processing and Power BI for visualization. Created a structured database to manage and visualize LEED certification trends over time.

 

Tools Used:

SQL (SQL Server Management Studio), Power BI, DAX, Excel.

 

Key Achievements:

  • Designed and implemented a relational database schema based on the business problem of understanding LEED certification trends.
  • Performed data extraction, transformation, and loading (ETL) to clean and normalize data from multiple sources, focusing on handling complex certification data.
  • Wrote complex SQL queries to extract insights, including views for certification level trends, project type distributions, and GFA impacts.
  • Connected Power BI to the SQL database to create interactive dashboards showcasing project certifications, certification levels, and their correlation with GFA.
  • Utilized DAX for advanced metrics and formatting to ensure dynamic reporting.

 

Outcome:

Provided comprehensive, data-driven visualizations to assess LEED certification performance, allowing users to interact with project types, certifications, and other key variables.
